# Action Loop Mock

This is a static, dependency-free VitalTrace prototype for testing the product advantage beyond lab dashboards.

The mock demonstrates this loop:

```text
red markers
-> focus areas
-> lifestyle experiment context
-> retest plan
-> after report comparison
-> proof of marker movement
```

## How To Open

Open `index.html` directly in a browser.

No server, package install, API keys, or build step is required.

## Demo Data

The prototype uses realistic anonymized sample values for markers such as HbA1c, LDL-C, hs-CRP, ALT, vitamin D, triglycerides, and HDL-C.

It does not use personal medical data.

## Safety Boundary

The mock is non-diagnostic and non-prescriptive.

It does not:

- Diagnose a condition.
- Recommend medication or treatment.
- Prescribe supplement doses.
- Replace clinician review.
- Promise that a lifestyle change caused a biomarker movement.

The intended product behavior is to help a serious self-directed health tracker compare reviewed reports, record lifestyle context, prepare retesting, and see whether markers moved after an action phase.
